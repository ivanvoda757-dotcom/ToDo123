# To-Do Lista (JavaScript verzija)

Jednostavna to-do lista napravljena u JavaScript-u sa HTML i CSS. Zadaci se čuvaju u browser-u (localStorage), pa ostaju i posle osvežavanja stranice.

## Kako pokrenuti lokalno:
1. Otvorite `index.html` u browser-u (dvokliknite fajl ili drag-and-drop u browser).

## Funkcionalnosti:
- Dodavanje zadataka
- Označavanje kao završeno/nije završeno
- Brisanje zadataka
- Uređivanje opisa zadatka
- Progress bar sa procentom završenih zadataka

## Za GitHub Pages:
1. Uploadujte ovaj folder na GitHub repo.
2. Idite na Settings > Pages > Source: Deploy from a branch > main (ili master).
3. Sajt će biti dostupan na `https://username.github.io/repo-name/`.

## Fajlovi:
- `index.html`: Glavna stranica
- `style.css`: Stilovi
- `script.js`: JavaScript logika
- `README.md`: Ovaj fajl
